NEW WORKOUT BUILDER FEATURES
- Click any exercise already added to the workout.
- An Exercise Details editor opens.
- Edit:
  • Exercise name
  • Written description
  • Coaching cues
  • Sets
  • Reps
  • Tempo
  • Rest period
  • Weight guidance
- Save the exercise details before assigning the workout.
- Details are stored inside the workout assignment in Supabase.
- Clients see the full details in their Workouts section.
